Created: 2026-06-04T22:04:00+07:00

## Overview
Xây dựng website Bảo Bối AI (baoboiai.com) - hệ sinh thái học liệu số kết hợp GEMS, NotebookLM và Vibe Coding. Phiên bản MVP sẽ tập trung vào việc chia sẻ tài nguyên (worksheets, slides, kịch bản prompts), nhúng các game web tương tác trực quan để học sinh chơi trực tiếp (Play Zone), chia sẻ mã nguồn game, và thu thập thông tin giáo viên (lead generation).


## Tech Stack
- Frontend: Next.js (React) + TailwindCSS
- Backend: Next.js Route Handlers (API routes)
- Database: Supabase (PostgreSQL + Object Storage) hoặc SQLite (local)
- Deployment: Vercel
